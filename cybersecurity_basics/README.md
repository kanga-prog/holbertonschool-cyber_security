# Cybersecurity Basics

This repository contains the foundational exercises and scripts for the **Cybersecurity Basics** project at Holberton School.  

## Overview

This project covers the essential concepts and practical exercises in cybersecurity, including:

- Introduction to Cybersecurity
- Ethical Hacking Basics
- Setting up and using a Cybersecurity OS (Kali Linux)

## Learning Objectives

By the end of this project, you should be able to explain:

- What is Cybersecurity?
- Core principles of cybersecurity (CIA Triad)
- How encryption contributes to security
- Risk management in cybersecurity
- Different types of cybersecurity threats
- The difference between a virus and a worm
- Social engineering in the context of security
- Key components of an information security program
- How security policies and frameworks contribute to an organization’s security posture
- Purpose of the OWASP Top Ten
- Role of access control in cybersecurity
- How multi-factor authentication enhances security
- Common methods for securing a network

## Requirements

- All files will be run on **Kali Linux 2023.2**
- Allowed editors: `vi`, `vim`, `emacs`
- All scripts must start with `#!/bin/bash`
- All scripts should end with a new line
- Scripts must be less than 2 lines (`wc -l file <= 2`)
- Do **not** use backticks, `&&`, `||`, or `;`
- Code must follow **Betty style**
- Do **not** use `printf`
